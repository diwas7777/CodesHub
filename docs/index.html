How to Contribute

Join our community and help make CodesHub even better. Every contribution matters!

1

Fork the Repository

Start by forking the CodesHub repository to your own GitHub account to create your personal copy.

2

Create a Branch

Create a new branch with a descriptive name for your contribution (e.g., add-python-sorting-algorithms).

3

Make Changes

Add your code examples, ensure they're well-commented, and follow the language-specific best practices.

4

Submit Pull Request

Submit a pull request with a clear description of your changes and wait for review from our maintainers.
